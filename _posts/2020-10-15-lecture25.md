# 석차 구하기

### 내 코드

```c
#include <stdio.h>

int main()
{

    int n;
    scanf("%d", &n);

  // 점수 입력 받을 배열
    int score[n];
  
  // 석차 누적 배열
    int rank[n];
  
    for (int i = 0; i < n; i++)
    {
        scanf("%d", &score[i]);
        rank[i] = 1;
    }

    for (int i = 0; i < n; i++)
    {
        for (int j = 0; j < n; j++)
        {
            if (i == j)
                continue;
//		값이 더 적으면 해당하는 인덱스의 rank에 누적
            else if (score[i] < score[j])
            {
                rank[i]++;
            }
        }
    }

    for (int i = 0; i < n; i++)
    {
        printf("%d ", rank[i]);
    }

    return 0;
}
```
